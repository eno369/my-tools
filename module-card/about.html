一枚のカードにまとめるツールです。

モジュールカードについて

「モジュールカード」は、ネドじゅん著『AI時代をどう生きる 右脳オカンの未来予言』（永岡書店）で紹介されているコンセプトをもとにしています。

AIが思考を担う時代に、人間が自分らしく輝くために「自分の好き・得意・在り方」を6つの要素として言語化し、カードとして可視化するものです。

6つの要素

🌬 風
好き・止められてもやりたいこと。原始的な衝動。
💎 玉
技術・能力・経験。社会で磨いてきたこと。
🤝 ハンド
他者に手を差し伸べられるスキルや役割。
🫀 コア
変わらない本質・クセ・自分の在り方。
🌌 空
他者に委ねる余白・苦手・手放すこと。
✨ コーリング
5要素から導かれる、魂からの問い・生涯の探求。

使い方

  1. ジェネレータを開き、名前を入力する
  2. 各要素の質問に答える（AIアシストあり・なし選べます）
  3. 6要素すべて入力すると「コーリング」が自動で導出される
  4. カードをプレビューして PNG で保存・共有

※ AIアシストを使う場合は Gemini API キー（無料枠あり）が必要です。なしでも手動入力で使えます。
