🔑 Gemini APIキーの取得方法

Googleアカウントがあれば、5〜10分で無料で取得できます。
初めての方向けに、画面の操作を一つずつ説明します。

✅ 費用について Gemini APIには無料枠があります。個人でカードを作る程度の利用なら、料金は発生しません。クレジットカードの登録も不要です。
⚠️ APIキーの取り扱いについて APIキーは「パスワードのようなもの」です。SNSや他の人に見せる画面に貼り付けないよう注意してください。このジェネレータはキーをお使いのスマホ・PCのみに保存します（外部には送信しません）。

手順

1

Google AI Studio を開く

以下のリンクをタップして、Google AI Studio を開きます。

https://aistudio.google.com

Googleアカウントでのログインを求められたら、普段使っているGmailのアカウントでログインしてください。

2

「APIキーを取得」をクリック

ページが開いたら、左側のメニューから 「Get API key」（APIキーを取得）をクリックします。

📱 画面のイメージ 左メニューに Get API key というボタンがあります。
見当たらない場合は画面左上の「≡」メニューを開いてみてください。
3

「APIキーを作成」をクリック

「Create API key」（APIキーを作成）という青いボタンをクリックします。

📱 画面のイメージ Create API key というボタンが表示されます。
クリックすると「プロジェクトを選択」という画面が出ることがあります。
その場合は 「Create API key in new project」 を選んでください。
4

APIキーをコピーする

文字と数字が混ざった長い文字列（APIキー）が表示されます。右側にある「コピー」ボタンをクリックしてコピーしてください。

📱 画面のイメージ AIza●●●●●●●●●●●●●●●●●●●●●●●●●●●●

このような形式の文字列です。「AIza」で始まる39文字です。
右側のコピーアイコンをクリックするとコピーできます。
5

ジェネレータに貼り付ける

モジュールカードジェネレータを開き、最初の画面（セットアップ画面）にある「Gemini APIキー」の入力欄に貼り付けます。

貼り付け方：入力欄を長押し（スマホ）または右クリック → 「貼り付け」（PC）

貼り付けたら「はじめる」ボタンをタップして完了です。

よくある質問

Q. Googleアカウントを持っていない場合は？

Gmailを使っていれば同じアカウントでログインできます。アカウントがない場合は こちらから無料で作れます。

Q. AIなしモードで使いたい場合は？

セットアップ画面で「AIなしモードで使う」にチェックを入れれば、APIキーなしでも使えます。質問のヒントを見ながら自分で入力する形式になります。

Q. キーを間違えて入力してしまったら？

セットアップ画面に戻れば入力し直せます。アプリの設定アイコンから「設定をリセット」を選んでください。
